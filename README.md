# testArchive

Анализ бинарных файлов .UJRNL внутри архивов .tar.gz
Этот проект предназначен для автоматизированного анализа бинарных файлов формата .UJRNL, которые находятся внутри множества архивов .tar.gz.
Скрипт извлекает первые байты каждого файла, проверяет их корректность, сравнивает с эталонной сигнатурой и формирует отчёт о состоянии данных.
